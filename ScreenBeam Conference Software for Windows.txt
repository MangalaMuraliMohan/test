Minimum system requirements

Windows 10 version 20H2 (Windows 11 support beginning in ScreenBeam Conference version 1.0.3.0)
Processor: Intel Core i5 (7th generation) quad core 1.6 GHz or AMD-equivalent
Memory: 8GB of RAM
Space: 200MB
Note: Microsoft Visual C++ redistributable package is included

Wirelessly connects to the in-room video conferencing USB peripherals
